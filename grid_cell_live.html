The cell is three velocity-controlled oscillators 120° apart, each running at f = 8 Hz + β·(v·d̂) — moving raises its frequency above the θ reference (panel 2). Where all three agree the cell fires, and those agreement points tile space into a hexagon (panel 1, painted by the spikes as the animal forages). Because the cell only fires when θ lines up with the local spatial phase, the firing phase is pinned to where the animal is, not to when it arrives — panel 3 holds a tight diagonal: position sets phase. That is phase precession, and it is the same beat as the checkerboard ECG, now carrying position instead of an accident. Notice that raising speed grows the VCO offset (panel 2) but the position→phase lock barely moves — the code is about where, not how fast. Built from the Burgess–O'Keefe oscillatory-interference model; numbers are measured live, the lock typically holds within ±10°.
