\# The Agreement Layer: Foveated Saccadic Perception

This folder contains the implementation of the **\*\*Agreement Layer\*\*** architecture—a visual perception engine that models how the brain merges a slow, blurry internal world model (Theta) with high-frequency reality (Gamma) using error-driven saccades and phase-coherence gating.

\!\[Architecture Diagram\](foveated\_agreement\_layer\_architecture.png)

\#\# The Theory: Why This Isn't Just "Blurry Video"

Modern AI tries to render reality as a panoramic 4K image. The biological brain doesn't. 

This system proves that human-like visual sharpness is an illusion sustained by an active **\*\*Agreement Layer\*\***. The architecture operates on three levels:  
1\. **\*\*The Gist (Theta / Prior):\*\*** A persistent, low-resolution, blurry internal prediction of the world. (In this demo, it is generated by a 35MB VAE trained on CelebA faces).  
2\. **\*\*The Retina (Gamma / Reality):\*\*** The raw, sharp, high-frequency photon data hitting your webcam.  
3\. **\*\*The Agreement Layer (The Gate):\*\*** A transient mathematical mask. Sharpness is *\*not stored\**. When the Fovea looks at a specific point, if the low-frequency prediction phase-locks with the high-frequency reality, the "gate" opens, and sharp detail flashes into existence for exactly one millisecond. 

When the prediction *\*fails\** (a phase mismatch), the system generates "Surprise." This surprise instantly calculates the gradient of the error and drives the next **\*\*Saccade\*\***—forcing the "eye" to dart to the anomaly to resolve it.

## **How to Run It**

### **1\. Download the Pre-Trained Face Model**

The 35MB SplatVAE face-prior model weights (model.pt) are hosted on HuggingFace:  
👉 [**Download model.pt from Aluode/Neuro\_Splat**](https://www.google.com/search?q=https://huggingface.co/Aluode/Neuro_Splat/tree/main)  
Place model.pt in the same directory (or note its path).

### **2\. Run the Live Saccade Engine**

Execute the script from your terminal. **You must pass the \--image\_size 128 and \--num\_packets 512 flags** to match the architecture of the downloaded CelebA model.

Bash  
python live\_agreement\_cortex.py \--cam 0 \--model model.pt \--num\_packets 512 \--image\_size 128

*(If your webcam fails to open on \--cam 0, try \--cam 1 or \--cam 2, especially if you use OBS).*

### **Fallback Modes (Testing without the Model)**

If you want to test the raw math of the Coherence Gate and error-driven saccades without loading the neural network, the script has built-in fallbacks that use a simple Gaussian Blur as the "Prior":

* **Test the Webcam (Zero Model):** python live\_agreement\_cortex.py \--cam 0  
* **Test on a Static Image:** python live\_agreement\_cortex.py \--image face.jpg  
* **Smoke Test (No Webcam, No Model):** python live\_agreement\_cortex.py \--smoke

## **What You Are Seeing**

When you run the live script, you will see four panels:

1. **Retina (Gamma):** The raw webcam feed.  
2. **Gist (Theta):** The output of the 35MB VAE. It is trying to force your webcam geometry onto its learned "Face Manifold." Notice it is blurry, generalized, and identity-agnostic, but accurately tracks your 3D head pose based entirely on 2D edge orientation.  
3. **Percept (Gist \+ Agreement):** This is the output. Notice it refuses to build a sharp panorama. It remains blurry everywhere *except* along the active scanpath, where the Agreement Layer briefly admits high-frequency reality.  
4. **Surprise \-\> Saccade:** The error map. The bright spots are where the Prior failed to predict Reality. The little circles show the mathematical "eye" constantly hunting the highest-error regions.

## **The Offline Proofs (Why "Surprise" matters)**

Also included in this folder is saccade\_agreement\_demo.py, a pure Numpy/CPU script that formally measures this phenomenon.  
**Experiment 1 (The Transient Layer):** Proves that the system actively refuses to store a sharp picture. The global percept barely improves over the blur (+1.08 dB). Sharpness is strictly local and transient.  
**Experiment 2 (The Wrong Regime):** Proves that "Surprise" only functions when the brain has an established baseline. If the prior is totally blurry, *every* edge looks like a surprise, and the eye wanders aimlessly. But when the Prior settles into agreement with the scene, the Phase Gate locks onto novel anomalies with devastating precision (7.71x enrichment), proving the eye is drawn to *timeline violations*, not just pixels.
